### <div align="center">FRONTEND</div>

# Requerimientos

```bash
$ node --version
v18.12.0

$ npm --version
8.19.2
```

# Instalacion  

```bash
$ git clone https://github.com/teamSilabuz/projectunidad_frontend
$ cd projectunidad_frontend
$ npm install
```

# Start & watch

```bash
$ npm start
```
> Nota: Es necesario crear tu archivo `.env`
> * Tomar como referencia el archivo .env.example
# Produccion

```bash
$ npm run build
```
